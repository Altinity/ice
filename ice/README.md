# ice-rest-catalog

A CLI for loading data into Iceberg REST catalogs (with ability to be used as ClickHouse UDF handler).

## Status

- [x] Configuration via `.ice.yaml` (when present) & env vars prefixed with `ICE_`.
- [x] s3:// support.
- [ ] gs:// support.
- [ ] az:// support.
